# exo-mlir

An experimental MLIR backend for the Exo programming language.

## Requirements

To build the benchmarks, you need the following dependencies:

-   `clang`
-   `ccache`
-   `lld`
-   `ninja`

Running `make` will build the required LLVM dependencies.
